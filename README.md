# 클라우드 네이티브 개발자 양성과정 - 백엔드 미니 프로젝트

## 농작물 모의 투자 사이트
> 현시점의 농작물을 주식처럼 사고 파는 모의 투자 사이트
> 기간 : 2021.07.19~2021.07.23  
> 사용한 기술 : 
>* Spring Boot  
>* Java 
>* CSS  
>* HTML
>* OracleDB

## 기능
> * 메인 페이지  
> * 로그인/회원가입 페이지
> * 농산물 상세 페이지  
>   + 
> * 실현 손익 페이지  
>   + 
> * 기사 페이지  
>   + 사이트에서 취급하는 농산물에 대한 기사를 크롤링하여 보여줌
